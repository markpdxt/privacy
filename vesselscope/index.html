VesselScope Privacy Policy

Effective Date: March 30, 2026

Overview

VesselScope is a marine vessel tracking app developed by PDXT, LLC. We respect your privacy. This policy describes what data the app accesses, how it is used, and what is shared with third parties.

We do not collect, store, or transmit any personal data to PDXT, LLC servers. VesselScope contains no advertising, no user accounts, and no telemetry beyond the anonymous analytics described below.

Data Used on Your Device

The following data is processed locally on your device and is never sent to PDXT, LLC:

  • Location (When In Use / Always): Your device coordinates are used to query for nearby vessels and to display your position on the map. If you enable background proximity alerts, the app requests "Always" location permission to continue monitoring traffic while backgrounded. Location data is not stored persistently.
  • API Credentials: If you use the AISStream.io marine data source, your API key is stored securely in the device keychain. It is never transmitted to PDXT, LLC.
  • App Preferences: Settings such as your selected data sources, filter criteria, proximity alert configuration, map zoom level, and API usage counters are stored locally on your device.
  • Vessel Cache: Recently fetched vessel data is temporarily cached in memory (30 seconds) and on disk (5 minutes) to reduce network requests. Cached data is automatically deleted when it expires.

Analytics

VesselScope uses Google Firebase Analytics to collect anonymous, aggregated usage data. This helps us understand how the app is used and improve the experience. Firebase Analytics may collect:

  • Device Information: Device model, operating system version, and app version.
  • Usage Data: App opens, screen views, and general interaction patterns.
  • Anonymous Identifiers: A random device identifier for aggregation (not linked to your identity).

Firebase Analytics does not collect your location, API keys, vessel data, or any personally identifiable information. No advertising identifiers are used. You can learn more about how Google handles this data in Firebase's Privacy Documentation.

Third-Party Services

Marine Vessel Data Sources

VesselScope connects to one of two marine vessel data sources, chosen by you:

  • AISStream.io: If you select the AISStream.io data source, your device's approximate geographic bounding box is sent via WebSocket to AISStream.io (stream.aisstream.io) to receive real-time vessel positions. Your AISStream API key is included for authentication. No other personal data is transmitted. This communication is subject to AISStream.io's terms.
  • Local AIS Receiver: If you select a local AIS receiver, VesselScope connects directly to a URL on your local network that you configure. No data leaves your local network and no third-party services are contacted.

Washington State Ferries

VesselScope may connect to the Washington State Department of Transportation (WSDOT) Ferries API to display ferry locations and schedules. Only publicly available ferry data is retrieved. No personal data is transmitted in these requests.

Data Sharing

We do not sell, rent, share, or transfer any user data to third parties for advertising, marketing, or any other purpose. The only network communication is between your device and the data sources you configure, plus anonymous analytics as described above.

Data Retention

VesselScope does not store any data on PDXT servers. All data resides on your device:

  • Memory cache: cleared after 30 seconds
  • Disk cache: cleared after 5 minutes
  • Preferences and settings: persist until you delete the app
  • API keys: stored in the device keychain until you delete the app

Anonymous analytics data collected by Firebase is retained according to Google's data retention policies.

Your Rights

Since all personal data remains on your device, you have full control:

  • Revoke location permission at any time in iOS Settings
  • Delete all stored data by deleting the app
  • Clear the vessel cache from within the app

Children's Privacy

VesselScope does not knowingly collect any personal information from children under 13. The app does not collect personal information from any user.

Changes to This Policy

We may update this policy from time to time. Changes will be reflected by updating the effective date above.
